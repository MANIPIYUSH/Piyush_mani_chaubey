# Piyush Mani Chaubey — Portfolio

Personal portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Live site

https://manipiyush.github.io/Piyush_mani_chaubey/

## Development

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` automatically deploy via GitHub Actions to the `gh-pages` branch.

If the site does not load, enable GitHub Pages once:

1. Open **Settings → Pages**
2. Set **Source** to **Deploy from a branch**
3. Choose branch **`gh-pages`** and folder **`/ (root)`**
4. Save

## Production build

```bash
VITE_BASE_PATH=/Piyush_mani_chaubey/ npm run build
```
